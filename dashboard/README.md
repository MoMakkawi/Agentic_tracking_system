# Agentic Tracking System - Dashboard

The **Agentic Tracking System Dashboard** is a high-performance, responsive web application designed for academic administrators and advisors to monitor orchestrated multi-agent workflows, inspect attendance logs, explore student cohorts, and manage security alerts.

Built with **React 18** and **Vite**, it features a premium, glassmorphism-inspired dark user interface ("Deep Ocean / Obsidian Theme") with interactive visualizations.

## Overview

The dashboard acts as the visual consumer of the FastAPI backend service. It directly integrates with the orchestrator agent and displays real-time results from the data pipeline, validation checks, and clustering models.

---

## Key Features

- **Real-time Overview**: Displays key telemetry, alert distributions, active student counts, and system status metrics.
- **Interactive Cohort Clustering**: Displays graph-based community groupings generated by the Louvain clustering agent, complete with continuity signals and cohort memberships.
- **Attendance Insights**: Interactive session history and drill-down details mapping session check-ins to scheduled iCalendar events.
- **Triaged Alerts System**: Separate tabs for **Device**, **Identity**, and **Timestamp** anomalies, mapping complex validation failures to readable tables.
- **Agent Chat Interface**: Allows users to interact directly with the central Orchestrator agent using natural language, returning responses and streaming updates.
- **Premium UI/UX**: frosted glass effects, custom HSL-tailored colors, smooth micro-animations, and responsive layouts.

## Getting Started

### Prerequisites

Ensure you have the following installed on your machine:

- **Node.js**: v18.0.0 or higher
- **npm**: v9.0.0 or higher

### Installation

1. Navigate to the dashboard directory:
   ```bash
   cd dashboard
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the local development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Building for Production

To create an optimized production build:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

## Styling Architecture

The project implements a modern CSS-in-CSS architecture located in `src/index.css`:

- **CSS Variables**: Core tokens defining background ranges (`--bg-primary` to `--bg-tertiary`), state colors (`--accent-success`, `--accent-warning`, `--accent-danger`), and accent glow effects.
- **Glassmorphism**: A system utility (`.glass`) that applies background-blur filters, semi-transparent borders, and drop-shadows to panels and buttons.
- **Typography**: Imports and applies Google Fonts: `Outfit` for display headings and `Inter` for body elements.
- **Animations**: CSS animations for keyframe entries (`fadeIn`, `scaleIn`) and smooth state transitions.
